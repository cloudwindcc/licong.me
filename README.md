# 李聪个人学术主页网站 (licong.me)

这是复旦大学信息科学与工程学院副教授李聪老师的个人学术主页网站，基于现代Web技术栈构建，展示其学术成就、研究方向和教学信息。

## 🎯 项目概述

本项目是一个现代化的学术个人主页网站，专为复旦大学李聪副教授设计，用于展示其：
- 个人简介与学术背景
- 复杂网络理论与应用研究方向
- 已出版的学术专著与论文
- 学术任职与社会服务
- 主导课程与教学成果
- 最新学术动态与获奖信息

## 🛠️ 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式框架**: Tailwind CSS
- **UI组件**: Headless UI + Lucide React图标库
- **动画效果**: tsParticles粒子动画系统
- **代码质量**: ESLint + TypeScript严格模式

## 🚀 快速开始

### 环境要求
- Node.js 16+
- npm 或 pnpm

### 安装与运行

```bash
# 克隆项目
git clone https://github.com/cloudwindcc/licong.me.git
cd licong.me

# 安装依赖
npm install

# 开发模式启动
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview

# 代码检查
npm run lint
```

## 🎨 设计特色

- **响应式设计**: 完美适配桌面、平板和移动设备
- **现代化UI**: 采用卡片式布局、毛玻璃效果和渐变背景
- **交互体验**: 粒子动画背景、悬停效果和折叠面板
- **双语展示**: 中英文双语学术信息展示
- **学术风格**: 专业、简洁、信息层次清晰

## 📊 内容模块

1. **个人简介**: 教育背景、研究方向、学术成就
2. **学术任职**: 学会任职、会议任职、期刊审稿
3. **主导课程**: 本科基础课、专业选修课、研究生课程
4. **学术专著**: 已出版书籍、购买链接
5. **期刊论文**: 代表性学术论文展示
6. **学术动态**: 最新获奖信息、学术活动
7. **相关链接**: 实验室、学术主页、社交平台

## 🌐 部署说明

项目已配置为静态网站部署，支持以下平台：

- **GitHub Pages**: 直接部署到github.io域名
- **Netlify**: 零配置部署，支持自定义域名
- **Vercel**: 一键部署，全球CDN加速
- **自建服务器**: 构建后部署到任何静态文件服务器
