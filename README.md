# 李聪个人学术主页网站 (licong.me)

这是复旦大学信息科学与工程学院副教授李聪老师的个人学术主页网站，基于现代Web技术栈构建，展示其学术成就、研究方向和教学信息。

## 🎯 项目概述

本项目是一个学术个人主页网站，专为复旦大学李聪副教授设计。

### 📋 核心内容模块
- **个人简介**: 教育背景、研究方向、学术成就
- **学术任职**: 学会任职、会议任职、期刊审稿
- **主导课程**: 本科基础课、专业选修课、研究生课程
- **学术专著**: 已出版书籍《复杂网络传播理论》
- **期刊论文**: 代表性学术论文（2015-2024年）
- **相关链接**: 实验室、学术主页、社交平台

## 🛠️ 技术栈

- **框架**: React 18 + TypeScript
- **构建**: Vite (极速开发体验)
- **样式**: Tailwind CSS (原子化CSS)
- **动画**: tsParticles粒子动画系统
- **图标**: Lucide React图标库
- **组件**: Headless UI无障碍组件

## 🚀 快速开始

### 环境要求
- **Node.js**: 18+
- **npm**: 8+ 或 pnpm

### 安装与运行

```bash
# 克隆项目
git clone https://github.com/cloudwindcc/licong.me.git
cd licong.me

# 安装依赖
npm install

# 开发模式启动
npm run dev

# 构建生产版本
npm run build
```
